Mappings from common file formats to RDF, and other considerations

Abstract

This document contains reference examples of types files with their schemas known to be used for exchanging data with developers of tools around the ecosystem of OpenStreetMap with their conterpairt with one or more encodings of Semantic Web standards.

Instead of explaining each kind of format, the intent here is to be a very superficial, example-oriented, quick reference between each format. Readers are recommended to look for the [[[#references]]].

Use https://respec.org/docs/ as reference for markup.
